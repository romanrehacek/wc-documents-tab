== Description ==

Adds new tab into a product detail page with downloadable documents.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/wc-documents-tab` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
